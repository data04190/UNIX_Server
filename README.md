# UNIX_Server
<br>

### 디렉토리 관련 명령어
<br>

|명령어|설명|
|:---:|---|
|pwd|현재 작업 디렉토리 출력|
|cd [디렉토리] |디렉토리로 이동|
|cd ~ |홈 디렉토리로 이동 |
|cd .. |부모 디렉토리로 이동 |
|ls |파일 및 디렉토리 리스트|
|ls -a |모든 파일 및 디렉토리 리스트|
|ls -asl |모든 파일 자세히 리스트|
|mkdir |디렉토리 만들기 |
|rmdir |디렉토리 삭제 |
<br>

### 파일 관련 명령어
<br>

|명령어|설명|
|:---:|---|
|cat [파일명]|파일 내용 출력|
|wc [파일명] |줄/단어/문자 수 출력|
|cp 파일1 파일2 |파일1 파일2로 복사 |
|rm [파일명]|파일 삭제 |
